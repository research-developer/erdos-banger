# Ralph Wiggum Archives

This directory stores archived `PROMPT.md` / `PROGRESS.md` snapshots from completed Ralph Wiggum sprints.

Rationale:
- keeps the root `PROMPT.md` / `PROGRESS.md` focused on the *next* run
- preserves historical context (what the loop was instructed to do and what it did)
- avoids “reward hacking” by making prompt/history auditable in git
